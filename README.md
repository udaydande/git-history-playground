# Git History Playground

This repository demonstrates how to read, inspect and reason about Git history.

## Undoing Changes Safely

- Git reset is used for local, unpushed commits.
- Git reset is used for public, shared history.
- Never rewrite history on main.

## Git Techniques Demonstarted

- Reading history with git log and git Balme.
- Safely undoing changes with git reset and revert.
- Cleaning feature branch history with rebase.
- Following professional git workflows.
